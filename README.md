
# 🏭  Factory Production and labour Analysis

This project analyzes production data from a garments factory using Microsoft Excel, focusing on performance tracking and data-driven insights.

---

## 📌 Project Overview
The goal is to track production performance, identify inefficiencies, and build a dynamic reporting dashboard for daily, weekly, and monthly insights.

## 🎯 Objectives
* **Analyze:** Target vs. Actual production metrics.
* **Measure:** Identify and visualize the production gap.
* **Calculate:** Efficiency (%) across different lines.
* **Visualize:** Build an interactive dashboard using Pivot Tables and slicers.

## 📊 Key Metrics (KPIs)
| Metric | Formula |
| :--- | :--- |
| **Production Gap** | `Target - Actual` |
| **Efficiency (%)** | `(Actual / Target) * 100` |

## 🛠 Tools Used
* **Microsoft Excel (Excel Online)**
* Pivot Tables & Slicers
* Data Cleaning Functions (`TRIM`, `CLEAN`, `DATEVALUE`)

## 🧹 Data Cleaning Process
Several data issues were identified and resolved:
1. **Date Standardization:** Fixed mixed text/numeric formats.
2. **Data Sanitization:** Used `TRIM()` and `CLEAN()` to remove hidden characters.
3. **Consistency:** Created helper columns for validation.
